Facebook App Publishing Checklist

Use these public URLs in the Meta app dashboard before switching the app to Live.

1. Create the app

  1. Go to Meta for Developers and create a new app.
  2. Choose a business use case if available.
  3. Add the Messenger product if you need inbound inquiries.

2. Fill in required app fields

  • Privacy Policy URL: /privacy-policy.html
  • Terms of Service URL: /terms-of-service.html
  • Data Deletion Instructions URL: /data-deletion.html
  • Support URL or contact: /support.html

3. Messenger webhook setup

  1. Create a public HTTPS webhook endpoint.
  2. Set a private verify token that matches your server environment.
  3. Subscribe to messages and messaging_postbacks.
  4. Use the Page access token only on the server.

4. Practical notes

  • Meta needs public, reachable URLs. Localhost will not work for app review.
  • Keep legal pages simple, accurate, and public.
  • Do not place the Page access token in the frontend or in `VITE_` env variables.
